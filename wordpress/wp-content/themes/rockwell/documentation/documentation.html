TimThumb Image Resizing Script - Common Problems - top

  1. !!! READ ONLY IF THE IMAGES YOU PERSONALLY INSERTED INTO THE SLIDER OR YOUR POST ARE NOT SHOWING UP !!!
  2. TimThumb will automatically create a /cache/ subfolder with proper write-permissions. If your host server doesn’t allow this by default, be sure to manually change the /cache/ folder permissions to 777. If you don't know how to change permissions, google: change permissions + "name of your FTP application"
  3. If neccessary, make sure the folder which contains TimThumb is also set to 777 as well.
  4. Please note that on some hostings you need to set 755 instead of 777.
  5. Please note that if everything above fails, you need to contact your hosting provider to white-list timthumb script
  6. Please note that this is not theme's issue. So please refer to your hosting providers - if they have a good support, they should know what you want, straight away

Category Manager - top

  1. Apply the same settings to all subcategories
  2. Select a category to edit
  3. Post Meta Settings
  4. Choose category template here





- Creating the Blog Category - top

  1. Create new category called 'Blog' - go to Post -> Categories and create the new category here
  2. Add new Blogpost to category 'Blog' - go to Post -> Add New, create a Post and don't forget to add this Post into your new Blog category!
  3. Select template in Category Manager
    • go to Theme Options -> Category Manager
    • Click the "Blog" button
    • Focus on 'Category view' templates
    • Find Select Post Type: and click at Blog
    • Find Select Sidebar and click at what sidebar you want
    • Click at the template name (for example blog-cat-1)
    • Focus on 'Single Post view' templates
    • Click at the template name (for example blog-single-1)
    • Hit the Save button

- Creating the Portfolio Category - top

  1. Create new category called 'Portfolio' - go to Post -> Categories and create the new category here
  2. Add new Post to category 'Portfolio' - go to Post -> Add New, create a Post and dont forgot to add this Post into your new Portfolio category!
  3. Select template in Category Manager
    • go to Theme Options -> Category Manager
    • Click the "Portfolio" button
    • Focus on 'Category view' templates
    • Find Select Post Type: and click at Portfolio
    • Click at the template name (for example portfolio-cat-1)
    • Focus on 'Single Post view' templates
    • Click at the template name (for example blog-single-1)
    • Hit the Save button

- Creating the Post - top

  1. Please note that single Posts (doesn't matter if it's Blog Post or Portfolio Post) have more than 20 templates. So for example one of your Portfolio Posts can use a Blog Single template. This is a very universal approach and we are hoping you will like it.
  2. You can adjust Single Post Template in:
    • Category Manager Theme Options > Category Manager, find "'Single Post view' templates" at the bottom of the page
    • Post Write Panels Posts > Add New, find Post Template click Customize and choose your template
    • !! If you select Post Template in Write Panel, it will have higher priority than Post Template from Category Manager
  3. - Customizing Post Meta
    • Go to Theme Options> Category Manager
    • Here you can customize your post meta settings, for single page and for category page
    • Go to FreshPanel > Translate > Post Meta
    • Here you can translate all your post meta data
